## How To Use

1. Copy the `.hh3` game file into the **root directory** of your calculator  
   (do not place it inside a subfolder).  

2. Launch the calculator and open the game through Hollyhock Launcher.  

3. Controls:  
   - Player 1: `‚Üê` / `‚Üí`  
   - Player 2: `1` / `3`  
   - `5` = restart game  
   - `EXE` or `Clear` = quit
